# 🍽️ Restaurant Sales Analysis

## 📌 Project Overview
This project analyzes restaurant sales data to identify customer purchasing patterns, revenue trends, product performance, and operational insights. The analysis was performed using Excel, SQL, Python, and Power BI to transform raw data into meaningful business insights and interactive dashboards.

## 🎯 Objective
The objective of this project is to:
- Analyze restaurant sales performance.
- Identify top-performing products and categories.
- Understand customer ordering behavior.
- Track revenue trends.
- Provide data-driven recommendations to improve business performance.

## 🛠️ Tools & Technologies
- Microsoft Excel (Data Cleaning & Preprocessing)
- SQL (Data Extraction & Business Queries)
- Python (Pandas, NumPy, Matplotlib for EDA)
- Power BI (Interactive Dashboard & Visualization)

## 📂 Dataset
The dataset contains restaurant transaction records including:
- Order Date & Time
- Product Name
- Category
- Quantity Sold
- Revenue
- Profit
- Payment Mode
- Customer & City Information

## 🔄 Project Workflow
1. Data Cleaning using Excel and Python
2. SQL queries for business analysis
3. Exploratory Data Analysis (EDA) using Python
4. KPI calculation and trend analysis
5. Interactive dashboard development in Power BI
6. Business recommendations based on insights

## 📊 Dashboard Features
- Total Revenue
- Total Profit
- Total Orders
- Average Order Value
- Monthly Sales Trend
- Category-wise Sales
- Top Selling Products
- City-wise Performance
- Time-wise Sales Analysis
- Payment Mode Analysis

## 💡 Key Insights
- Identified the highest revenue-generating product categories.
- Analyzed monthly and seasonal sales trends.
- Compared sales performance across different cities.
- Evaluated customer purchasing behavior by time of day.
- Identified the most profitable products and categories.

## 📈 Business Recommendations
- Focus marketing efforts on high-performing product categories.
- Introduce promotional offers during low-sales periods.
- Optimize inventory based on product demand.
- Improve customer engagement through targeted offers.
- Expand operations in high-performing locations.
